# GetKindleMemos
This is a Bookmarklet to get memos from your Kindle.

## Usage
1. Add this javascript code to bookmarklet.
2. Open the your "Kindle memo".
	https://read.amazon.co.jp/notebook
3. Click the bookmarklet.
4. Paste it to your note.
